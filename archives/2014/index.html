​Linux中npm出现npmlog找不到的解决方法

##描述

今天在对nodejs项目进行服务器迁移的时，在新的linux服务器上输入 npm -v 命令后报 “cannot find module ‘npmlog’….”的错误。找到的原因是拷贝过去的bin目录下的npm文件并没有链接到module目录下的npm-cli.js文件，所以导致在执行 npm -v 命令时，路径问题导致npmlog模块无法找到。
